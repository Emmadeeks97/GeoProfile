# Convert lat-lon to bearing

Calculate bearing and great circle distance between an origin and one or
more destination points

## Usage

``` r
latlon_to_bearing(origin_lat, origin_lon, dest_lat, dest_lon)
```

## Arguments

- origin_lat:

  latitude of origin point

- origin_lon:

  longitude of origin point

- dest_lat:

  latitude of destination point

- dest_lon:

  longitude of destination point

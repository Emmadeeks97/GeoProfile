# Cholera data

Locations of cases of cholera from the 1854 cholera outbreak made famous
by John Snow.

## Usage

``` r
data(Cholera)
```

## Format

an object of class `list`.

## Examples

``` r
if (FALSE) { # interactive()
plot(Cholera$latitude, Cholera$longitude, xlab = "lon", ylab = "lat")
}
```
